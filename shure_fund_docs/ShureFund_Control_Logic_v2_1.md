# Control Logic v2.1 (Updated)

## Override Types

### Treasury Override

-   Performed by funder/treasury
-   Can bypass blockers
-   Must record overriddenBlockers
-   Must log reason

### Contractor Supplementary Contribution

-   Contractor adds funds to cover shortfall
-   Funds tagged as contractor-origin
-   Restricted to specific stage/work package
-   Must not be treated as general funder funds

## Core Rules

-   No work without funding
-   No approval without evidence
-   No payment without approval
-   All actions logged
